Данный скрипт парсит выгрузку из каналов и чатов телеграм. Когда вы экспортируете историю чата, создается один или несколько файлов html. Вот их и обрабатывает данный скрипт. Если вы будете выгружать все файлы: видео, аудио и прочие, то они будут перемещены в отдельные директории video и files, а ссылки на них добавлены в текст полученный со страницы. Затем скрипт сохраняет полученные данные в формате DOCX, для их возможной дальнейшей обработки. Требования по библиотекам в файле requirements.txt.
-----------------------------------------------------
This script parses the upload from telegram channels and chats. When you export the chat history, one or more html files are created. So this script processes them. If you upload all the files: video, audio, and others, they will be moved to separate video and files directories, and links to them will be added to the text received from the page. The script then saves the received data in DOCX format for possible further processing. Library requirements in the file requirements.txt.
